# Running

## Requires Scraper

Install scraper dependencies and run.

## Installation

`pip install whoosh`

Create dir `indexdir`

## Running

For one Algorithm `python search.py ALG CON TERMS`

For all Algorithms `python searchCompare.py CON TERMS`

ALG = BM25F, Frequency, TF_IDF

CON = OR, AND

TERMS = *anything*
